### Лабораторная работа по Вычислитеной математике.

Данная программа аппроксимирует заданный набор точек при помощи:
1. Полинома Лагранжа;
2. Метода наименьших квадтратов (полиномы до 3ей степени);

и выводит данные функции на экран при помощи SFML.

Так же программа находит площади методом Симпсона.

#### Структура входного файла
Файл(*.txt), содеражащий в себе точки, выглядит следующим образом:
- 1 строчка - процент выхода за границы;
- 2 строчка - нужно ли считать полином Лагранжа;
- 3 строчка - количество точек;
- 4 и далее - x y.
